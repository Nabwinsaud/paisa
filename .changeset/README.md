# Changesets

Hello and welcome! This folder has been automatically generated by `@changesets/cli`, a build tool that works
with multi-package repos, or single-package repos to help you version and publish your code. You can
find the full documentation for it [in the readme](https://github.com/changesets/changesets/blob/main/README.md)

## Usage

### Creating a changeset

```bash
bun changeset
```

This will prompt you to select which packages have changed, and whether the change is a major, minor, or patch bump.

### Versioning

```bash
bun changeset version
```

This consumes all pending changesets, bumps package versions, and writes changelogs.

### Publishing

```bash
bun changeset publish
```

This publishes all packages that have unpublished versions to npm.
